## How to Take Screenshots

### Desktop (Windows)
- Use `Win + Shift + S` to open Snipping Tool
- Or use browser DevTools (F12) and select device frame

### Full Page Screenshots
- Chrome: DevTools → Cmd+Shift+P → "Capture full size screenshot"
- Firefox: Right-click → "Take Screenshot" → "Save full page"

### Mobile Screenshots
- Open DevTools (F12)
- Toggle device toolbar (Ctrl+Shift+M)
- Select a mobile device (iPhone 12 Pro recommended)
- Take screenshot

## Recommended Dimensions
- Desktop: 1920x1080 or 1440x900
- Mobile: 390x844 (iPhone 12 Pro)
